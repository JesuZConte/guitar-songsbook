# ADR-014: App Icon Design — Guitar Pick with Eighth Note

## Status
Deferred — accepted for v1, revision considered for v2

## Context
The current icon is a guitar pick shape (cream `#FFF8E1`) with an eighth note (brown `#4E342E`) on a warm brown background. It was designed to convey the music/guitar identity of the app quickly.

After v1 launch it was noted that the icon has visual similarity to Guitar Tuna (a popular tuner app), which also uses a warm colour palette and a music-related symbol on a pick-like shape. While the designs are not identical and there is no confirmed trademark conflict, the resemblance is close enough to warrant a redesign before the app grows in visibility.

## Decision
Keep the current icon for v1 to avoid delaying the release. Revisit for v2.

## Options for v2

1. **Keep pick shape, change symbol** — replace the eighth note with a different mark (e.g. a chord diagram grid, staff lines, or the letter C for Cancionero). Low effort, maintains brand recognition.

2. **New shape entirely** — move away from the pick. Options: an open book with a musical note, a notebook/journal motif (reinforces the "personal repertoire" angle), or a stylised "C".

3. **Commission a professional designer** — now that the app has a proper name (Cancionero) and a clear identity (personal chord songbook), a designer can build a distinctive icon around that concept.

## Recommendation for v2
Option 3 is preferred if budget allows. Option 1 is the fastest safe fix. Avoid colours and shapes that overlap with Guitar Tuna (amber/warm brown + pick silhouette).

## Consequences
- No action required before Play Store submission
- Flag for v2 planning: icon redesign should be one of the first visual tasks
